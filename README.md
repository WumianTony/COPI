# [COPI] Course of Python Introduction

# 课程标题：Python 开发入门

### 课程目标
让你逐步拥有独立学习、独立 Debug、独立开发的能力，基于 Python，但不止于 Python。

### 课程对象
- 任何想学习 Python 的朋友
- 需要一定数学理解能力
- 需要一些英语词汇基础

### 课程时长
- 新手建议慢慢来，急不得
- 有其他语言编程基础可以加速
- 有 Python 基础可以跳过学过的章节

### 教学方法
课上
1. 通过 Runoob / Jupyter Notebook 的资料学习
2. 完成 Jupyter Notebook 上的样例
课下
1. 完成洛谷指定题目
2. 有任何问题欢迎联系无冕进行答疑
